# Quick Start Guide

## Run the Application

1. **Install dependencies** (if not already done):
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Open your browser**:
   Navigate to the URL shown in the terminal (usually `http://localhost:5173`)
